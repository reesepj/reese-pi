---
name: handoff
description: Handoff an active CLI session or work context to Telegram so the user can continue from their phone.
---

The user wants to handoff their current work from a desktop/CLI Pi session to continue on Telegram from their phone.

Please do the following:

1. Acknowledge the handoff request
2. Summarize the current context / last prompt / active work if available
3. Confirm that the session state is now available in this Telegram conversation
4. Offer to continue exactly where they left off (e.g. “Ready to continue working on the landing page?”)
5. Keep the response short and clear for mobile

This handoff allows the user to get up from their desk and keep working on the same task via Telegram.
